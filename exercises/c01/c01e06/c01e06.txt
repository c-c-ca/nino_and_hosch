Examples of applicable queries and the appropriate return values for the following objects:

a. classroom in a class scheduling program - query: roomNumber(), type of return value: int

b. book in a library catalogue - query: title(), type of return value: String

c. back pack in a retail inventory system - query: itemNumber(), type of return value: int

d. back pack in a video game - query: color(), type of return value: reference-to-Color
                               *Color is a class defined separately by the software designer

e. elevator in a simulation - query: load(), type of return value: double
